# My Skills

Personal collection of AI Agent skills for coding workflow.

## Skills

| Skill | Description |
|-------|------------|
| `dev-flow` | Full-cycle development workflow: plan, diagnose, implement, verify, sync, ship |

## Installation

Copy the skill directory to `~/.codex/skills/` or your agent's skills directory.

```bash
cp -r dev-flow ~/.codex/skills/
```
